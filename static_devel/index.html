To see normal instructions, look HERE
The development version may contain bugs, including ones that can break whole BCX, meaning you might loose some of your BCX settings or even have to fully reset the data while using it. Use it at your own risk!

The following methods will keep the BCX version constantly up-to-date without any need to install it again. BCX is open source.

Manual activation using bookmark:

1) Drag the following link to your bookmark bar: BCX (Development version)
2) Click the bookmark while in Club (or login screen)

Manual activation using bookmark: (Android chrome version)

1) Long-press the following link and select "Copy link address": BCX (Development version)
2) Press 3 dots -> Bookmark star -> Edit (on bottom of the screen in popup)
3) Long-tap text in URL (or select all) and paste copied text
4) Leave bookmark menu using back arrow in top left corner
To activate:
1) Enter Club
2) Tap URL bar
3) Write the name of the bookmark you saved
4) Tap it (It will have star on the left)
*) Do not try selecting the bookmark from list of bookmarks, it won't work!
